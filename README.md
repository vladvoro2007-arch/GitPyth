# Отчёт по лабораторной работе №1 

Этот проект представляет собой результат выполнения вводной лабораторной работы, в ходе которой было настроено рабочее окружение и изучены базовые инструменты разработчика.

---

##  Часть 1. Jupyter 

Первый этап работы был посвящён знакомству с интерактивной средой разработки **Jupyter Notebook**.
### Выполненные шаги:
* **Запуск среды**: Использовалась веб-версия через официальный сайт `jupyter.org` с выбором опции "Try it in your browser".
*  **Интерфейс**: Работа велась в среде **JupyterLab** — интерактивном интерфейсе нового поколения для работы с блокнотами и кодом.
*  **Работа с библиотеками**: Согласно заданию, был установлен модуль `arrow`.
*  **Выполнение кода**: Была вызвана функция `arrow.now()`, с помощью которой текущее время было выведено в консоль блокнота.

**Пример кода из блокнота:**
```python
import arrow
print(arrow.now())
```
