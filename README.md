> Мощный и современный инструмент на Python, созданный с упором на чистый код, надежную архитектуру и автоматизацию рутины. Проект вдохновлен передовыми практиками разработки open-source решений.



\---



\## ✨ Особенности (Features)



\* \*\*Современная архитектура\*\*: Использование `src-layout` для изоляции исходного кода и предотвращения ошибок импорта.

\* \*\*Автоматическое версионирование\*\*: Интеграция `setuptools-git-versioning` позволяет автоматически генерировать версии релизов на основе Git-тегов и истории коммитов.

\* \*\*Идеальный код\*\*: Строгое соблюдение стандартов форматирования PEP 8 благодаря встроенной поддержке \*\*Black\*\* (с лимитом в 88 символов на строку).

\* \*\*Удобная сборка\*\*: Вся конфигурация проекта, зависимостей и плагинов хранится в едином файле `pyproject.toml`.
